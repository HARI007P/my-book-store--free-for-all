

# 📚 Hari's BookStore

Hari's BookStore is a responsive, full-stack web application designed to help book lovers browse, donate, and pay for books seamlessly. Built using **React** (Frontend), **Node.js + Express** (Backend), and **MongoDB** (Database), the platform also supports **UPI Payments**, **order verification**, and **SMS/email notifications**.

## 🚀 Features

- ✅ Browse available books
- ✅ UPI payment with screenshot upload
- ✅ Secure order form with UTR validation
- ✅ SMS/email reminders to collect orders
- ✅ Donation section to contribute old books
- ✅ Timer for payment submission with voice alerts
- ✅ Fully responsive design with Tailwind CSS
- ✅ Animated background and image integration

## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS, React Router, React Toastify
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)
- **Other Tools:** Axios, Nodemailer, Twilio, FormData, HTML5 SpeechSynthesis
